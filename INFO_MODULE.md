# 📊 Модуль Info

Модуль для отображения информации о юзерботе Forelka.

## 🎯 Команда

```
.info
```

## 📋 Отображаемая информация

- 🔥 **Название** - Forelka Userbot
- 👤 **Владелец** - Имя и фамилия из профиля Telegram
- 🌿 **Branch** - Текущая ветка Git
- ⚙️ **Prefix** - Текущий префикс команд
- ⏱ **Uptime** - Время работы юзербота с момента запуска
- 💾 **RAM usage** - Использование оперативной памяти (требуется psutil)
- 🖥 **Host** - Имя хоста/сервера

## 🔧 Зависимости

- `pyrogram` / `kurigram` - основной фреймворк
- `psutil` (опционально) - для отображения использования памяти

Если `psutil` не установлен, вместо использования RAM будет отображаться "N/A".

## 📦 Установка зависимостей

```bash
pip install -r requirements.txt
```

## ⚙️ Технические детали

### Как работает uptime

Время старта сохраняется в `client.start_time` при запуске юзербота в `main.py`:

```python
await client.start()
client.start_time = time.time()
```

Модуль вычисляет разницу между текущим временем и временем старта.

### Получение информации о владельце

Используется `client.me` - объект, который доступен после `await client.start()`.

### Формат вывода

Используется HTML форматирование с blockquote и custom emoji:

```html
<blockquote>
<emoji id=5461117441612462242>🔥</emoji> Forelka Userbot
<blockquote>
...
</blockquote>
</blockquote>
```

## 🐛 Устранение проблем

### Модуль не загружается

Проверьте:
1. Файл `modules/info.py` существует
2. Синтаксис файла корректен: `python3 -m py_compile modules/info.py`
3. Модуль регистрируется: проверьте функцию `register()`

### RAM usage показывает "N/A"

Установите `psutil`:
```bash
pip install psutil
```

### Uptime всегда показывает 0 секунд

Убедитесь, что в `main.py` добавлена строка:
```python
client.start_time = time.time()
```

После `await client.start()` и перед `loader.load_all(client)`.
